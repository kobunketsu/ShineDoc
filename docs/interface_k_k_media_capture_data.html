◆ replaceDepthDataForPhoto:

- (nullable AVDepthData *) replaceDepthDataForPhoto: (AVCapturePhoto *)  photo

替换深度信息

Parameters
photo<#photo description#>

改造系统拍摄到的深度图 主要方式：将拍摄前的深度信息作为中值滤镜的参考输入对系统深度图进行中值滤波，用来平滑表面3x3噪波。对后期计算平滑的法线移除尖刺非常有利。 拍摄的深度图和拍摄前实时预览的深度图如果分辨率，格式，属性都不相同，需要匹配后在改造。 ref 尺寸可能不相同。例如video:3088x2311，photo:3088x2316. intrinsicMatrix也会有微笑的不同 从拍摄模式>拍摄输出环境(光线，稳定性等)>输出数据精准度>输出深度格式，做改造的要求限制
