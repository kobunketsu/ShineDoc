- (KKWatermarkConfiguration *) requestWatermarkConfig

获得一个水印配置拷贝

通过选择器的方式，配合媒体标签获得对应的水印配置

根据照片信息特效选出候选配置标签，比例根据人头位置可剔除遮挡人脸的配置

‍已经取消了水印不出现在人脸上的设定，完全随机，如果出现在人脸上，那就是迫使用户付费购买的时机

随机选择一个可能出现的位置

没有找到就使用默认水印配置 必须拷贝才能修改
