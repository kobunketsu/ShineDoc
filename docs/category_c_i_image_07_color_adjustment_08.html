◆ imageByApplyingGamma:

- (CIImage *) imageByApplyingGamma: (CGFloat)  gamma

gamma

Parameters
gamma<gamma description#>

This filter is typically used to compensate for nonlinear effects of displays. Adjusting the gamma effectively changes the slope of the transition between black and white. It uses the following formula: pow(s.rgb, vec3(power)) NOTE As with all color filters, this operation is performed in the working color space of the Core Image context (CIContext) executing the filter, using unpremultiplied pixel color values. If you see unexpected results, verify that your output and working color spaces are set up as intended.
