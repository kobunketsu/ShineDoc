◆ cameraZoomMax

- (CGFloat) cameraZoomMax
readnonatomicassign

如果是具有深度信息，人像遮罩的镜头，并且使用深度功能，则缩放不超过系统提供的最大zoom的2倍 XR的后置镜头最高分辨率默认有深度信息，但风景模式不使用深度

带深度的镜头只在拍照的情况下支持非原生的数字缩放，视频输出不走cropFilter不能支持非原生缩放 注意后置镜头提供深度的分辨率普遍低 XR: 400x300，XS: 320x240, 缩放到2倍导致使用中最高只能提供1/2的精度，portrait mode can still use ai to generate data.
